# Solución rápida de Windows Update – Windows 7

1. `services.msc`  
2. Transferencia Inteligente en Segundo Plano ~ Automático ~ Reiniciar  
3. Windows Update ~ Automático ~ Reiniciar  
4. `C:\Windows\SoftwareDistribution` ~ Borrar todas las carpetas  
5. Cifrado ~ Deshabilitar temporalmente  
6. `C:\Windows\System32\catroot2` ~ Borrar carpeta  
7. Cifrado ~ Automático ~ Reiniciar  
8. [Catálogo Microsoft Update KB3138612](https://www.catalog.update.microsoft.com/Search.aspx?q=KB3138612)  
9. Reiniciar equipo
